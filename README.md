# Chicago Transit Ridership vs Air Quality Analysis

## Project Overview
This project analyzes the relationship between Chicago public transit ridership (CTA) and air pollution levels (PM2.5). The objective is to explore whether changes in transit usage correlate with variations in air quality.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code
- Tableau

## Data Sources
- Chicago Transit Authority (CTA) Ridership Data
- Chicago Air Quality PM2.5 Monitoring Data

## Project Workflow
1. Data collection from CTA ridership dataset and air quality dataset
2. Data cleaning and preprocessing using Python (Pandas)
3. Monthly aggregation of ridership and PM2.5 values
4. Correlation analysis between ridership and air pollution
5. Visualization using Python
6. Dashboard creation using Tableau

## Key Visualizations
- Monthly PM2.5 Pollution Trend
- Monthly CTA Ridership Trend
- Ridership vs Pollution Scatter Plot
